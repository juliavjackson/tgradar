# Инструкция по передаче проекта на другой аккаунт

Передача проекта «Telegram Stats Bot» состоит из основных этапов: передача прав на бота, настройка API-ключей, передача кода и перенос контекста Antigravity.

---

## 1. Передача прав на Telegram-бота

Чтобы сменить владельца бота:
1. Откройте [@BotFather](https://t.me/BotFather).
2. Выберите бота -> **Bot Settings** -> **Transfer Ownership**.
3. Следуйте инструкциям (требуется 2FA включенная >7 дней).

---

## 2. Настройка Telegram API (API ID и API Hash)

Новый владелец должен получить свои ключи на [my.telegram.org](https://my.telegram.org) и обновить их в файле `.env`.

---

## 3. Сброс сессии (Важно!)

Удалите файл `anon.session`. При первом запуске новый владелец авторизуется под своим номером телефона.

---

## 4. Передача в Antigravity

1. **Код проекта**: Передайте всю папку с проектом.
2. **Артефакты и контекст**: Я (Antigravity-агент) сохранил текущие инструкции и задачи в папке `docs/antigravity/`. 
3. **Для нового владельца**: После открытия проекта в Antigravity, попросите агента: *"Прочитай документацию в папке docs/antigravity, там инструкции по передаче и текущие задачи"*. Это поможет ИИ быстро войти в курс дела.

---

## 5. Обновление .env

Обновите `BOT_TOKEN`, `TELEGRAM_API_ID`, `TELEGRAM_API_HASH` и `TELEGRAM_PHONE`.

---

## 6. Передача кода

Используйте Git (Transfer ownership на GitHub) или просто zip-архив (без `venv` и `anon.session`).
